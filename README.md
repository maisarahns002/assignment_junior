# PHP Electricity Calculator

This project is a simple PHP (vanilla) application to calculate:

- Power (Wh)
- Energy (kWh)
- Total Electricity Charge (RM)

## Formula Used

Power (Wh) = Voltage (V) × Current (A)

Energy (kWh) = Power × Hour ÷ 1000

Total Charge (RM) = Energy × (Rate ÷ 100)

## Features

- User input for voltage, current, and electricity rate
- Calculate electricity usage per hour (1–24 hours)
- Display result in table format
- Bootstrap UI design

## Technology Used

- PHP (Vanilla)
- HTML
- Bootstrap 4
